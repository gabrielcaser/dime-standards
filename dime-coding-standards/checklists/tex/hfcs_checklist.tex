		\section*{High-frequency checks}
  This checklist lists important factors to consider while reviewing your code review partner's \textbf{high-frequency checks}. Please fill this checklist, and submit it as an attachment when you submit \href{https://survey.wb.surveycto.com/collect/code_review_summary?caseid=}{this detailed form.}

  \hfill \break 
  \textbf{Note:} We are piloting this checklist as part of improving the code review process. The questions in this checklist are therefore \textbf{not required}, and we welcome your feedback on the points listed below.

  \subsection*{General}
		
		\begin{itemize}
			\setlength\itemsep{-0.1em}
			\item[] \dimeCheckBox{The data shared does not contain any personally-identifying information (PII).}

                \item[] \dimeCheckBox{The code creates a tracking sheet (or dashboard) to track progress of the survey.}
        \end{itemize}

 \hfill \break 
The following sections cover various aspects of the \textbf{high-frequency checks} that we recommended for ensuring high quality data, but not all may apply to the high-frequency checks you are reviewing.

\subsection*{Survey progress}
Tracking \textbf{survey progress} involves checks for the following:
            \begin{itemize}

            \item \textbf{Response rates -}
            These include checks/summary stats for:
            
            \begin{itemize}
            
                \item[] \dimeCheckBox{Number/share of surveys attempted, completed, and declined.}
                
                \item[] \dimeCheckBox{Number/share of surveys attempted, completed, and declined - for each day}
                
                \item[] \dimeCheckBox{Total surveys attempted, completed, and declined - on aggregate, and/or over last \textbf{X} days}
                
                \item[] \dimeCheckBox{Reasons for declining (categorical)}
                
            \end{itemize}


            \item\textbf{Duration -}
            These include checks/summary stats for:
            
            \begin{itemize}
            
                \item[] \dimeCheckBox{Average duration for each day for each interview}
                
                \item[] \dimeCheckBox{Duration by each module/section}
                
            \end{itemize}

            \end{itemize}
            

\subsection*{Survey quality}
Tracking \textbf{survey quality} involves checks for the following:
            \begin{itemize}
            
            \item\textbf{Duplicates -}
            These include checks/summary stats for:
            
            \begin{itemize}
            
                \item[] \dimeCheckBox{Duplicate entries in ID variables}

                \item[] \dimeCheckBox{Information to be used to resolve duplicates, such as listing non-ID variables for which duplicates has different values}
                
            \end{itemize}
            
            \item\textbf{Distribution of key variables:}
            These include checks/summary stats for:
            
            \begin{itemize}
            
                \item[]\dimeCheckBox{Missing values and outliers for all key variables} 
                
                \item[]\dimeCheckBox{Errors in the data that need to be shared with the field team for corrections}
                
                \item[]\dimeCheckBox{Distribution of \textbf{continuous variables}}
                
                \item[]\dimeCheckBox{Number of extended missing responses (like \texttt{Do not know}, \texttt{Refuse to answer}) by question type }
                
                \item[]\dimeCheckBox{Number of outliers by question type}
                
            \end{itemize}

            \item\textbf{Inconsistency in responses -}
            These include checks/summary stats for:
            \begin{itemize}
            
                \item[]\dimeCheckBox{Inconsistent responses with questions and their values that needs to be shared with the field team for corrections. For example, inconsistencies across variables for the same respondent, or for the same variables across different time periods.}
                
            \end{itemize}

            \item\textbf{Form version used -} 
            These include summary stats for:
            \begin{itemize}
            
                \item[]\dimeCheckBox{Number of responses by form-version and date}
                
            \end{itemize}

            \end{itemize}

\subsection*{Enumerator performance checks}
Tracking \textbf{enumerator performance} involves checks for the following:

            \begin{itemize}

            \item\textbf{Response rates -}
            These include checks/summary stats for:
            
            \begin{itemize}
            
                \item[] \dimeCheckBox{Number/share of surveys attempted, completed, and declined, for each enumerator}
                
                \item[] \dimeCheckBox{Number of attempts made to contact each interviewee, for each enumerator}
                
                \item[] \dimeCheckBox{Total surveys attempted, completed, and declined, for each enumerator}
                
            \end{itemize}


            \item\textbf{Duration -}
            These include checks/summary stats for:
            
            \begin{itemize}
            
                \item[] \dimeCheckBox{Average duration of the interview, for each enumerator}
                
                \item[] \dimeCheckBox{Average duration per module, for each enumerator}
                
            \end{itemize}


            \item\textbf{Missing data -}
            These include checks/summary stats for:
            
            \begin{itemize}
            
                \item[] \dimeCheckBox{Overall missing rate by question type, for each enumerator}
                
                \item[] \dimeCheckBox{Average missing rate for key variables (that might result in skipping an entire section) by each enumerator}
                
            \end{itemize}

            \item\textbf{Distribution/outliers -} 
            These include checks/summary stats for:
            
            \begin{itemize}
            
                \item[] \dimeCheckBox{Distribution for each key variable (including outliers) for each enumerator}
                
                \item[] \dimeCheckBox{Average number of outliers for each enumerator}
                
            \end{itemize}

            \item\textbf{Inconsistency in responses -}
            These include checks/summary stats for:
            
            \begin{itemize}
                
                \item[] \dimeCheckBox{Total number of inconsistent responses for each enumerator}

                \item[] \dimeCheckBox{Average number of inconsistent responses per day, for each enumerator}
                
            \end{itemize}

            \item\textbf{Flagged errors -}
            These include checks/summary stats for:
            
            \begin{itemize}

                \item[] \dimeCheckBox{Total flagged errors for each enumerator}

                \item[] \dimeCheckBox{Average number of flagged errors per day, for each enumerator}
                
            \end{itemize}

            \end{itemize}
